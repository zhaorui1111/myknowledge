# 15 · 动态规划（Dynamic Programming）详解

> **核心思想**：将复杂问题分解为相互重叠的子问题，通过记录子问题的解避免重复计算，自底向上（或自顶向下）构建最终答案。
>
> 动态规划（Dynamic Programming, DP）是算法设计中最强大、应用最广泛的范式之一。它不是某一个具体算法，而是一种**解题方法论**——只要问题满足「最优子结构」和「重叠子问题」两大性质，就可以用 DP 高效求解。

---

## 一、概念与直觉（What & Why）

### 1.1 什么是动态规划

动态规划的名字来自 Richard Bellman（1953），"Programming" 在此指「规划/决策」（类似 Linear Programming），而非编程。其核心在于：

- **分阶段决策**：将问题按阶段划分，每个阶段做出一个决策
- **状态记录**：用「状态」描述当前阶段的局面
- **状态转移**：通过「状态转移方程」从已知子问题推导出当前问题的解
- **避免重复计算**：通过「备忘录」或「表格」存储已算过的子问题答案

### 1.2 与分治、贪心的对比

| 特征 | 分治（Divide & Conquer） | 贪心（Greedy） | 动态规划（DP） |
|------|-------------------------|---------------|---------------|
| 子问题关系 | 子问题互不重叠 | 无需回顾子问题 | 子问题大量重叠 |
| 是否保证最优 | 是（合并阶段处理） | 不一定（局部最优≠全局） | 是（穷举所有子问题） |
| 求解方向 | 自顶向下递归 | 顺序贪心 | 自底向上 or 记忆化 |
| 典型例子 | 归并排序、快排 | 活动选择、Huffman | 背包、最长子序列 |

### 1.3 DP 能解决的问题特征

一个问题适合用 DP 求解，通常具备：

1. **最优子结构（Optimal Substructure）**：问题的最优解包含子问题的最优解。即可以通过子问题的最优解推导出原问题的最优解。
2. **重叠子问题（Overlapping Subproblems）**：递归求解时，同一个子问题会被重复计算多次。DP 通过存储子问题的解来消除重复。
3. **无后效性（Memorylessness）**：某阶段的状态一旦确定，后续决策只依赖当前状态，不受之前决策路径的影响。

### 1.4 直觉理解：Fibonacci 的演化

```
朴素递归：fib(5) 展开有大量重复
                    fib(5)
                   /      \
              fib(4)       fib(3)
             /     \       /    \
         fib(3)  fib(2) fib(2) fib(1)
         /   \
     fib(2) fib(1)

重复计算：fib(3) 算了 2 次，fib(2) 算了 3 次
```

**DP 的解法**：只算一次每个 fib(i)，存表复用。时间从 O(2^n) 降到 O(n)。

## 二、理论基础

### 2.1 最优子结构的证明方法

证明问题具有最优子结构通常用「剪切-粘贴」（Cut-and-Paste）反证法：

1. 假设问题 P 的最优解为 S*
2. S* 包含子问题 P' 的一个解 S'
3. 假设 S' 不是 P' 的最优解，即存在更好的解 S''
4. 用 S'' 替换 S* 中的 S'，得到更好的 P 的解——与 S* 是最优解矛盾
5. 因此 S' 必是 P' 的最优解 ✓

### 2.2 状态转移方程

DP 的核心是**状态转移方程**（Recurrence Relation），形如：

```
dp[i] = f(dp[i-1], dp[i-2], ..., dp[0])
```

设计步骤：
1. **定义状态**：dp[i] 或 dp[i][j] 代表什么含义
2. **建立转移**：dp[i] 如何由更小的子问题推导
3. **确定初始条件**：base case
4. **确定遍历顺序**：保证计算 dp[i] 时所依赖的状态已算好
5. **确定答案位置**：最终答案在 dp 表的哪里

### 2.3 两种实现范式

#### 自顶向下（Top-Down）+ 记忆化（Memoization）

```swift
// Swift
var memo: [Int: Int] = [:]

func fib(_ n: Int) -> Int {
    if n <= 1 { return n }
    if let cached = memo[n] { return cached }
    let result = fib(n - 1) + fib(n - 2)
    memo[n] = result
    return result
}
```

```objectivec
// Objective-C
@interface Solution : NSObject
@property (nonatomic, strong) NSMutableDictionary<NSNumber *, NSNumber *> *memo;
@end

@implementation Solution

- (NSInteger)fib:(NSInteger)n {
    if (n <= 1) return n;
    NSNumber *key = @(n);
    NSNumber *cached = self.memo[key];
    if (cached) return cached.integerValue;
    NSInteger result = [self fib:n - 1] + [self fib:n - 2];
    self.memo[key] = @(result);
    return result;
}

@end
```

#### 自底向上（Bottom-Up）+ 填表（Tabulation）

```swift
// Swift
func fib(_ n: Int) -> Int {
    if n <= 1 { return n }
    var dp = [Int](repeating: 0, count: n + 1)
    dp[0] = 0
    dp[1] = 1
    for i in 2...n {
        dp[i] = dp[i - 1] + dp[i - 2]
    }
    return dp[n]
}
```

```objectivec
// Objective-C
- (NSInteger)fib:(NSInteger)n {
    if (n <= 1) return n;
    NSInteger *dp = calloc(n + 1, sizeof(NSInteger));
    dp[0] = 0;
    dp[1] = 1;
    for (NSInteger i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    NSInteger result = dp[n];
    free(dp);
    return result;
}
```

### 2.4 空间优化：滚动变量 / 滚动数组

当 dp[i] 只依赖于有限个前驱状态时，可以将 O(n) 空间降为 O(1)：

```swift
// Swift - 空间 O(1)
func fib(_ n: Int) -> Int {
    if n <= 1 { return n }
    var prev2 = 0, prev1 = 1
    for _ in 2...n {
        let curr = prev1 + prev2
        prev2 = prev1
        prev1 = curr
    }
    return prev1
}
```

对于二维 DP，若 dp[i][j] 只依赖 dp[i-1][...]，可用两行滚动数组将 O(n*m) 降为 O(m)。

### 2.5 Top-Down vs Bottom-Up 对比

| 维度 | Top-Down（记忆化） | Bottom-Up（填表） |
|------|-------------------|------------------|
| 实现方式 | 递归 + 哈希表/数组缓存 | 循环 + 数组 |
| 子问题计算 | 按需计算（Lazy） | 全部计算 |
| 栈溢出风险 | 有（递归深度大时） | 无 |
| 空间优化 | 困难 | 容易（滚动数组） |
| 适用场景 | 状态空间稀疏、不确定哪些子问题需要 | 状态空间密集、需要空间优化 |
| 代码风格 | 更直觉、更接近递推公式 | 更高效、更工程化 |

## 三、DP 问题分类（Taxonomy）

动态规划问题可按「状态维度」和「问题类型」进行分类：

### 3.1 按状态维度分类

| 类型 | 状态定义 | 典型问题 |
|------|---------|---------|
| 线性 DP | dp[i] 表示前 i 个元素的最优解 | LIS、最大子数组和 |
| 区间 DP | dp[i][j] 表示区间 [i,j] 的最优解 | 矩阵链乘、戳气球、回文子串 |
| 背包 DP | dp[i][w] 表示前 i 物品容量 w 的最优 | 0/1背包、完全背包、多重背包 |
| 树形 DP | dp[node] 表示以 node 为根的子树的解 | 树直径、打家劫舍III |
| 状态压缩 DP | dp[mask] 用位掩码表示集合状态 | TSP、棋盘覆盖 |
| 数位 DP | 按数字位逐位决策 | 计数特定性质的数 |
| 概率/期望 DP | dp[i] 表示某状态的期望值 | 随机游走、骰子问题 |

### 3.2 五大经典 DP 模式

1. **线性序列型**：LIS / LCS / 编辑距离 / 最大子数组
2. **背包型**：0/1背包 / 完全背包 / 分组背包 / 多重背包
3. **区间合并型**：矩阵链乘 / 石子合并 / 戳气球
4. **路径网格型**：不同路径 / 最小路径和 / 三角形最小路径
5. **状态机型**：股票买卖系列 / 打家劫舍系列

## 四、线性 DP 经典问题

### 4.1 爬楼梯（Climbing Stairs）

**问题**：每次可爬 1 或 2 阶，到第 n 阶有多少种方法？

**状态**：dp[i] = 到第 i 阶的方法数
**转移**：dp[i] = dp[i-1] + dp[i-2]（最后一步要么爬1阶，要么爬2阶）
**初始**：dp[0] = 1, dp[1] = 1

```swift
// Swift
func climbStairs(_ n: Int) -> Int {
    if n <= 1 { return 1 }
    var prev2 = 1, prev1 = 1
    for _ in 2...n {
        let curr = prev1 + prev2
        prev2 = prev1
        prev1 = curr
    }
    return prev1
}
```

```objectivec
// Objective-C
- (NSInteger)climbStairs:(NSInteger)n {
    if (n <= 1) return 1;
    NSInteger prev2 = 1, prev1 = 1;
    for (NSInteger i = 2; i <= n; i++) {
        NSInteger curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

**复杂度**：时间 O(n)，空间 O(1)

### 4.2 最大子数组和（Maximum Subarray / Kadane's Algorithm）

**问题**：给定整数数组，找到和最大的连续子数组。

**状态**：dp[i] = 以 nums[i] 结尾的最大子数组和
**转移**：dp[i] = max(nums[i], dp[i-1] + nums[i])
**答案**：max(dp[0], dp[1], ..., dp[n-1])

```swift
// Swift - Kadane's Algorithm
func maxSubArray(_ nums: [Int]) -> Int {
    var maxEndingHere = nums[0]
    var maxSoFar = nums[0]
    
    for i in 1..<nums.count {
        maxEndingHere = max(nums[i], maxEndingHere + nums[i])
        maxSoFar = max(maxSoFar, maxEndingHere)
    }
    return maxSoFar
}
```

```objectivec
// Objective-C
- (NSInteger)maxSubArray:(NSArray<NSNumber *> *)nums {
    NSInteger maxEndingHere = nums[0].integerValue;
    NSInteger maxSoFar = maxEndingHere;
    
    for (NSUInteger i = 1; i < nums.count; i++) {
        NSInteger num = nums[i].integerValue;
        maxEndingHere = MAX(num, maxEndingHere + num);
        maxSoFar = MAX(maxSoFar, maxEndingHere);
    }
    return maxSoFar;
}
```

**复杂度**：时间 O(n)，空间 O(1)
**变体**：若需返回子数组下标，维护 start/end 指针即可。

### 4.3 最长递增子序列（Longest Increasing Subsequence, LIS）

**问题**：给定数组，找到最长严格递增子序列的长度。

#### 方法一：经典 DP — O(n²)

**状态**：dp[i] = 以 nums[i] 结尾的 LIS 长度
**转移**：dp[i] = max(dp[j] + 1) for all j < i where nums[j] < nums[i]
**答案**：max(dp[0..n-1])

```swift
// Swift - O(n²)
func lengthOfLIS(_ nums: [Int]) -> Int {
    let n = nums.count
    if n == 0 { return 0 }
    var dp = [Int](repeating: 1, count: n)
    var maxLen = 1
    
    for i in 1..<n {
        for j in 0..<i {
            if nums[j] < nums[i] {
                dp[i] = max(dp[i], dp[j] + 1)
            }
        }
        maxLen = max(maxLen, dp[i])
    }
    return maxLen
}
```

```objectivec
// Objective-C - O(n²)
- (NSInteger)lengthOfLIS:(NSArray<NSNumber *> *)nums {
    NSInteger n = nums.count;
    if (n == 0) return 0;
    NSInteger *dp = calloc(n, sizeof(NSInteger));
    for (NSInteger i = 0; i < n; i++) dp[i] = 1;
    NSInteger maxLen = 1;
    
    for (NSInteger i = 1; i < n; i++) {
        for (NSInteger j = 0; j < i; j++) {
            if (nums[j].integerValue < nums[i].integerValue) {
                dp[i] = MAX(dp[i], dp[j] + 1);
            }
        }
        maxLen = MAX(maxLen, dp[i]);
    }
    free(dp);
    return maxLen;
}
```

#### 方法二：贪心 + 二分 — O(n log n)

维护一个「tails」数组：tails[i] = 长度为 i+1 的 IS 的最小末尾元素。

```swift
// Swift - O(n log n) Patience Sorting
func lengthOfLIS(_ nums: [Int]) -> Int {
    var tails: [Int] = []
    
    for num in nums {
        // 二分查找 tails 中第一个 >= num 的位置
        var lo = 0, hi = tails.count
        while lo < hi {
            let mid = (lo + hi) / 2
            if tails[mid] < num {
                lo = mid + 1
            } else {
                hi = mid
            }
        }
        if lo == tails.count {
            tails.append(num)
        } else {
            tails[lo] = num
        }
    }
    return tails.count
}
```

```objectivec
// Objective-C - O(n log n)
- (NSInteger)lengthOfLIS:(NSArray<NSNumber *> *)nums {
    NSMutableArray<NSNumber *> *tails = [NSMutableArray array];
    
    for (NSNumber *numObj in nums) {
        NSInteger num = numObj.integerValue;
        NSInteger lo = 0, hi = tails.count;
        while (lo < hi) {
            NSInteger mid = (lo + hi) / 2;
            if (tails[mid].integerValue < num) {
                lo = mid + 1;
            } else {
                hi = mid;
            }
        }
        if (lo == (NSInteger)tails.count) {
            [tails addObject:@(num)];
        } else {
            tails[lo] = @(num);
        }
    }
    return tails.count;
}
```

**复杂度**：时间 O(n log n)，空间 O(n)

### 4.4 最长公共子序列（Longest Common Subsequence, LCS）

**问题**：给定两个字符串，求最长公共子序列的长度。

**状态**：dp[i][j] = text1[0..i-1] 与 text2[0..j-1] 的 LCS 长度
**转移**：
- 若 text1[i-1] == text2[j-1]：dp[i][j] = dp[i-1][j-1] + 1
- 否则：dp[i][j] = max(dp[i-1][j], dp[i][j-1])

```swift
// Swift
func longestCommonSubsequence(_ text1: String, _ text2: String) -> Int {
    let s1 = Array(text1), s2 = Array(text2)
    let m = s1.count, n = s2.count
    // dp[i][j]: LCS of s1[0..<i] and s2[0..<j]
    var dp = [[Int]](repeating: [Int](repeating: 0, count: n + 1), count: m + 1)
    
    for i in 1...m {
        for j in 1...n {
            if s1[i - 1] == s2[j - 1] {
                dp[i][j] = dp[i - 1][j - 1] + 1
            } else {
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
            }
        }
    }
    return dp[m][n]
}
```

```objectivec
// Objective-C
- (NSInteger)longestCommonSubsequence:(NSString *)text1 text2:(NSString *)text2 {
    NSInteger m = text1.length, n = text2.length;
    // 使用一维滚动数组优化空间
    NSInteger *prev = calloc(n + 1, sizeof(NSInteger));
    NSInteger *curr = calloc(n + 1, sizeof(NSInteger));
    
    for (NSInteger i = 1; i <= m; i++) {
        unichar c1 = [text1 characterAtIndex:i - 1];
        for (NSInteger j = 1; j <= n; j++) {
            unichar c2 = [text2 characterAtIndex:j - 1];
            if (c1 == c2) {
                curr[j] = prev[j - 1] + 1;
            } else {
                curr[j] = MAX(prev[j], curr[j - 1]);
            }
        }
        NSInteger *temp = prev;
        prev = curr;
        curr = temp;
        memset(curr, 0, (n + 1) * sizeof(NSInteger));
    }
    NSInteger result = prev[n];
    free(prev);
    free(curr);
    return result;
}
```

**复杂度**：时间 O(m×n)，空间可优化到 O(min(m,n))

### 4.5 编辑距离（Edit Distance / Levenshtein Distance）

**问题**：给定两个字符串 word1 和 word2，求将 word1 转换成 word2 所使用的最少操作数（插入/删除/替换）。

**状态**：dp[i][j] = word1[0..i-1] 转换为 word2[0..j-1] 的最少操作数
**转移**：
- 若 word1[i-1] == word2[j-1]：dp[i][j] = dp[i-1][j-1]（不需操作）
- 否则：dp[i][j] = 1 + min(dp[i-1][j-1], dp[i-1][j], dp[i][j-1])
  - dp[i-1][j-1] + 1：替换
  - dp[i-1][j] + 1：删除 word1[i-1]
  - dp[i][j-1] + 1：在 word1 中插入 word2[j-1]

**初始条件**：dp[i][0] = i（删 i 次），dp[0][j] = j（插 j 次）

```swift
// Swift
func minDistance(_ word1: String, _ word2: String) -> Int {
    let s1 = Array(word1), s2 = Array(word2)
    let m = s1.count, n = s2.count
    
    // 空间优化：只用一行 + 一个变量
    var dp = Array(0...n)  // dp[j] 对应上一行的 dp[i-1][j]
    
    for i in 1...m {
        var prev = dp[0]  // dp[i-1][j-1]
        dp[0] = i
        for j in 1...n {
            let temp = dp[j]
            if s1[i - 1] == s2[j - 1] {
                dp[j] = prev
            } else {
                dp[j] = 1 + min(prev, dp[j], dp[j - 1])
                // prev=替换, dp[j]=删除(上方), dp[j-1]=插入(左方)
            }
            prev = temp
        }
    }
    return dp[n]
}
```

```objectivec
// Objective-C
- (NSInteger)minDistance:(NSString *)word1 word2:(NSString *)word2 {
    NSInteger m = word1.length, n = word2.length;
    NSInteger *dp = malloc((n + 1) * sizeof(NSInteger));
    for (NSInteger j = 0; j <= n; j++) dp[j] = j;
    
    for (NSInteger i = 1; i <= m; i++) {
        NSInteger prev = dp[0];
        dp[0] = i;
        unichar c1 = [word1 characterAtIndex:i - 1];
        for (NSInteger j = 1; j <= n; j++) {
            NSInteger temp = dp[j];
            unichar c2 = [word2 characterAtIndex:j - 1];
            if (c1 == c2) {
                dp[j] = prev;
            } else {
                dp[j] = 1 + MIN(prev, MIN(dp[j], dp[j - 1]));
            }
            prev = temp;
        }
    }
    NSInteger result = dp[n];
    free(dp);
    return result;
}
```

**复杂度**：时间 O(m×n)，空间 O(min(m,n))

**应用场景**：拼写纠错、DNA 序列比对、版本 diff、模糊搜索

## 五、背包问题系列（Knapsack Problems）

背包问题是 DP 最经典的应用之一，衍生出多种变体。

### 5.1 0/1 背包（0/1 Knapsack）

**问题**：有 n 个物品，每个物品有重量 w[i] 和价值 v[i]，背包容量为 W。每个物品只能选 0 或 1 次，求最大总价值。

**状态**：dp[i][j] = 前 i 个物品、容量为 j 时的最大价值
**转移**：
- 不选第 i 个：dp[i][j] = dp[i-1][j]
- 选第 i 个（前提 j >= w[i]）：dp[i][j] = dp[i-1][j-w[i]] + v[i]
- 取两者较大值

**空间优化**：一维数组 + **逆序遍历容量**（保证每个物品只用一次）

```swift
// Swift - 0/1 背包（一维优化）
func knapsack01(weights: [Int], values: [Int], capacity: Int) -> Int {
    let n = weights.count
    var dp = [Int](repeating: 0, count: capacity + 1)
    
    for i in 0..<n {
        // 逆序遍历容量，确保每个物品只被使用一次
        for j in stride(from: capacity, through: weights[i], by: -1) {
            dp[j] = max(dp[j], dp[j - weights[i]] + values[i])
        }
    }
    return dp[capacity]
}
```

```objectivec
// Objective-C - 0/1 背包
- (NSInteger)knapsack01WithWeights:(NSArray<NSNumber *> *)weights
                            values:(NSArray<NSNumber *> *)values
                          capacity:(NSInteger)capacity {
    NSInteger n = weights.count;
    NSInteger *dp = calloc(capacity + 1, sizeof(NSInteger));
    
    for (NSInteger i = 0; i < n; i++) {
        NSInteger w = weights[i].integerValue;
        NSInteger v = values[i].integerValue;
        for (NSInteger j = capacity; j >= w; j--) {
            dp[j] = MAX(dp[j], dp[j - w] + v);
        }
    }
    NSInteger result = dp[capacity];
    free(dp);
    return result;
}
```

**复杂度**：时间 O(n×W)，空间 O(W)

> **注意**：0/1 背包是「伪多项式时间」——O(nW) 中 W 是数值大小而非输入位数，所以当 W 很大时并非高效。

### 5.2 完全背包（Unbounded Knapsack）

**问题**：每个物品可以选无限次。

**区别**：遍历容量时**正序**（允许重复选取同一物品）。

```swift
// Swift - 完全背包
func knapsackUnbounded(weights: [Int], values: [Int], capacity: Int) -> Int {
    let n = weights.count
    var dp = [Int](repeating: 0, count: capacity + 1)
    
    for i in 0..<n {
        // 正序遍历容量，允许物品被重复选取
        for j in weights[i]...capacity {
            dp[j] = max(dp[j], dp[j - weights[i]] + values[i])
        }
    }
    return dp[capacity]
}
```

```objectivec
// Objective-C - 完全背包
- (NSInteger)knapsackUnboundedWithWeights:(NSArray<NSNumber *> *)weights
                                   values:(NSArray<NSNumber *> *)values
                                 capacity:(NSInteger)capacity {
    NSInteger n = weights.count;
    NSInteger *dp = calloc(capacity + 1, sizeof(NSInteger));
    
    for (NSInteger i = 0; i < n; i++) {
        NSInteger w = weights[i].integerValue;
        NSInteger v = values[i].integerValue;
        for (NSInteger j = w; j <= capacity; j++) {
            dp[j] = MAX(dp[j], dp[j - w] + v);
        }
    }
    NSInteger result = dp[capacity];
    free(dp);
    return result;
}
```

### 5.3 背包问题变体总结

| 变体 | 物品使用限制 | 容量遍历顺序 | 典型应用 |
|------|------------|------------|---------|
| 0/1 背包 | 每个至多 1 次 | 逆序 | 子集和、分割等和子集 |
| 完全背包 | 每个无限次 | 正序 | 零钱兑换、完全平方数 |
| 多重背包 | 每个至多 k[i] 次 | 二进制拆分 → 0/1 | 库存分配 |
| 分组背包 | 每组选至多 1 个 | 逆序 + 组内枚举 | 互斥选择 |
| 混合背包 | 混合 0/1 和完全 | 按类型选策略 | 综合场景 |

### 5.4 经典应用：零钱兑换（Coin Change）

**问题**：给定面额数组 coins 和总金额 amount，求凑出 amount 的最少硬币数（每种面额无限）。

```swift
// Swift - 完全背包变体
func coinChange(_ coins: [Int], _ amount: Int) -> Int {
    var dp = [Int](repeating: amount + 1, count: amount + 1)
    dp[0] = 0
    
    for coin in coins {
        for j in coin...amount {
            dp[j] = min(dp[j], dp[j - coin] + 1)
        }
    }
    return dp[amount] > amount ? -1 : dp[amount]
}
```

```objectivec
// Objective-C
- (NSInteger)coinChange:(NSArray<NSNumber *> *)coins amount:(NSInteger)amount {
    NSInteger *dp = malloc((amount + 1) * sizeof(NSInteger));
    for (NSInteger i = 0; i <= amount; i++) dp[i] = amount + 1;
    dp[0] = 0;
    
    for (NSNumber *coinObj in coins) {
        NSInteger coin = coinObj.integerValue;
        for (NSInteger j = coin; j <= amount; j++) {
            dp[j] = MIN(dp[j], dp[j - coin] + 1);
        }
    }
    NSInteger result = dp[amount] > amount ? -1 : dp[amount];
    free(dp);
    return result;
}
```

**复杂度**：时间 O(n × amount)，空间 O(amount)

### 5.5 分割等和子集（Partition Equal Subset Sum）

**问题**：判断数组能否分成两个子集使它们的和相等。

**转化**：是否存在子集和为 sum/2 → 0/1 背包（容量 = sum/2，物品 = 数组元素）

```swift
// Swift
func canPartition(_ nums: [Int]) -> Bool {
    let sum = nums.reduce(0, +)
    if sum % 2 != 0 { return false }
    let target = sum / 2
    
    var dp = [Bool](repeating: false, count: target + 1)
    dp[0] = true
    
    for num in nums {
        for j in stride(from: target, through: num, by: -1) {
            dp[j] = dp[j] || dp[j - num]
        }
    }
    return dp[target]
}
```

```objectivec
// Objective-C
- (BOOL)canPartition:(NSArray<NSNumber *> *)nums {
    NSInteger sum = 0;
    for (NSNumber *n in nums) sum += n.integerValue;
    if (sum % 2 != 0) return NO;
    NSInteger target = sum / 2;
    
    BOOL *dp = calloc(target + 1, sizeof(BOOL));
    dp[0] = YES;
    
    for (NSNumber *numObj in nums) {
        NSInteger num = numObj.integerValue;
        for (NSInteger j = target; j >= num; j--) {
            dp[j] = dp[j] || dp[j - num];
        }
    }
    BOOL result = dp[target];
    free(dp);
    return result;
}
```

## 六、区间 DP（Interval DP）

### 6.1 概念

区间 DP 的特征：在一个序列上，以区间 [i, j] 作为状态，枚举分割点 k，将区间一分为二分别求解后合并。

**通用模式**：
```
dp[i][j] = optimal over k in [i, j-1]:
    combine(dp[i][k], dp[k+1][j]) + cost(i, k, j)
```

**遍历顺序**：按区间长度从小到大枚举（先算短区间，再算长区间）。

### 6.2 矩阵链乘法（Matrix Chain Multiplication）

**问题**：给定 n 个矩阵的维度，求最少乘法次数的括号化方案。

矩阵 A_i 的维度为 p[i-1] × p[i]，即 p 数组长度为 n+1。

**状态**：dp[i][j] = 计算矩阵 A_i...A_j 的最少乘法次数
**转移**：dp[i][j] = min(dp[i][k] + dp[k+1][j] + p[i-1]×p[k]×p[j]) for k in [i, j-1]
**初始**：dp[i][i] = 0

```swift
// Swift
func matrixChainOrder(_ p: [Int]) -> Int {
    let n = p.count - 1  // n 个矩阵
    if n <= 1 { return 0 }
    
    // dp[i][j] = 最少乘法次数 (1-indexed)
    var dp = [[Int]](repeating: [Int](repeating: 0, count: n + 1), count: n + 1)
    
    // 按链长度枚举
    for len in 2...n {
        for i in 1...(n - len + 1) {
            let j = i + len - 1
            dp[i][j] = Int.max
            for k in i..<j {
                let cost = dp[i][k] + dp[k + 1][j] + p[i - 1] * p[k] * p[j]
                dp[i][j] = min(dp[i][j], cost)
            }
        }
    }
    return dp[1][n]
}
```

```objectivec
// Objective-C
- (NSInteger)matrixChainOrder:(NSArray<NSNumber *> *)p {
    NSInteger n = p.count - 1;
    if (n <= 1) return 0;
    
    // 分配二维数组
    NSInteger **dp = calloc(n + 1, sizeof(NSInteger *));
    for (NSInteger i = 0; i <= n; i++) {
        dp[i] = calloc(n + 1, sizeof(NSInteger));
    }
    
    for (NSInteger len = 2; len <= n; len++) {
        for (NSInteger i = 1; i <= n - len + 1; i++) {
            NSInteger j = i + len - 1;
            dp[i][j] = NSIntegerMax;
            for (NSInteger k = i; k < j; k++) {
                NSInteger cost = dp[i][k] + dp[k + 1][j]
                    + p[i - 1].integerValue * p[k].integerValue * p[j].integerValue;
                dp[i][j] = MIN(dp[i][j], cost);
            }
        }
    }
    NSInteger result = dp[1][n];
    for (NSInteger i = 0; i <= n; i++) free(dp[i]);
    free(dp);
    return result;
}
```

**复杂度**：时间 O(n³)，空间 O(n²)

### 6.3 戳气球（Burst Balloons, LeetCode 312）

**问题**：n 个气球排成一排，戳破气球 i 获得 nums[left]×nums[i]×nums[right] 的硬币（left/right 为相邻未戳破气球），求最大硬币数。

**关键转化**：反向思考——不想「先戳哪个」，而是想「最后戳哪个」。

**状态**：dp[i][j] = 戳破开区间 (i, j) 内所有气球的最大硬币数
**转移**：dp[i][j] = max(dp[i][k] + dp[k][j] + nums[i]×nums[k]×nums[j]) for k in (i+1, j-1)

在两端添加虚拟气球 1，数组变为 [1, nums[0], ..., nums[n-1], 1]。

```swift
// Swift
func maxCoins(_ nums: [Int]) -> Int {
    let arr = [1] + nums + [1]
    let n = arr.count
    var dp = [[Int]](repeating: [Int](repeating: 0, count: n), count: n)
    
    // 区间长度从 2 开始（开区间内至少有 1 个气球）
    for len in 2..<n {
        for i in 0..<(n - len) {
            let j = i + len
            for k in (i + 1)..<j {
                let coins = arr[i] * arr[k] * arr[j] + dp[i][k] + dp[k][j]
                dp[i][j] = max(dp[i][j], coins)
            }
        }
    }
    return dp[0][n - 1]
}
```

```objectivec
// Objective-C
- (NSInteger)maxCoins:(NSArray<NSNumber *> *)nums {
    NSMutableArray<NSNumber *> *arr = [NSMutableArray arrayWithObject:@1];
    [arr addObjectsFromArray:nums];
    [arr addObject:@1];
    NSInteger n = arr.count;
    
    NSInteger **dp = calloc(n, sizeof(NSInteger *));
    for (NSInteger i = 0; i < n; i++) dp[i] = calloc(n, sizeof(NSInteger));
    
    for (NSInteger len = 2; len < n; len++) {
        for (NSInteger i = 0; i < n - len; i++) {
            NSInteger j = i + len;
            for (NSInteger k = i + 1; k < j; k++) {
                NSInteger coins = arr[i].integerValue * arr[k].integerValue * arr[j].integerValue
                    + dp[i][k] + dp[k][j];
                dp[i][j] = MAX(dp[i][j], coins);
            }
        }
    }
    NSInteger result = dp[0][n - 1];
    for (NSInteger i = 0; i < n; i++) free(dp[i]);
    free(dp);
    return result;
}
```

**复杂度**：时间 O(n³)，空间 O(n²)

### 6.4 最长回文子串（Longest Palindromic Substring）

**状态**：dp[i][j] = s[i..j] 是否为回文
**转移**：dp[i][j] = (s[i] == s[j]) && (j - i < 3 || dp[i+1][j-1])

```swift
// Swift - 区间 DP
func longestPalindrome(_ s: String) -> String {
    let chars = Array(s)
    let n = chars.count
    if n < 2 { return s }
    
    var dp = [[Bool]](repeating: [Bool](repeating: false, count: n), count: n)
    var start = 0, maxLen = 1
    
    // 所有单字符都是回文
    for i in 0..<n { dp[i][i] = true }
    
    // 按长度枚举
    for len in 2...n {
        for i in 0...(n - len) {
            let j = i + len - 1
            if chars[i] == chars[j] {
                dp[i][j] = (len <= 3) || dp[i + 1][j - 1]
            }
            if dp[i][j] && len > maxLen {
                start = i
                maxLen = len
            }
        }
    }
    let startIdx = s.index(s.startIndex, offsetBy: start)
    let endIdx = s.index(startIdx, offsetBy: maxLen)
    return String(s[startIdx..<endIdx])
}
```

> **更优解**：Manacher 算法可在 O(n) 时间内求解，但理解难度更高。中心扩展法也是 O(n²) 但常数更小。

## 七、路径与网格 DP（Grid DP）

### 7.1 不同路径（Unique Paths）

**问题**：m×n 网格，从左上到右下，只能向右或向下移动，有多少条路径？

**状态**：dp[i][j] = 到达 (i,j) 的路径数
**转移**：dp[i][j] = dp[i-1][j] + dp[i][j-1]

```swift
// Swift - 空间优化为一行
func uniquePaths(_ m: Int, _ n: Int) -> Int {
    var dp = [Int](repeating: 1, count: n)
    for _ in 1..<m {
        for j in 1..<n {
            dp[j] += dp[j - 1]
        }
    }
    return dp[n - 1]
}
```

```objectivec
// Objective-C
- (NSInteger)uniquePaths:(NSInteger)m n:(NSInteger)n {
    NSInteger *dp = malloc(n * sizeof(NSInteger));
    for (NSInteger j = 0; j < n; j++) dp[j] = 1;
    for (NSInteger i = 1; i < m; i++) {
        for (NSInteger j = 1; j < n; j++) {
            dp[j] += dp[j - 1];
        }
    }
    NSInteger result = dp[n - 1];
    free(dp);
    return result;
}
```

**复杂度**：时间 O(m×n)，空间 O(n)
**数学解法**：C(m+n-2, m-1) = (m+n-2)! / ((m-1)! × (n-1)!)

### 7.2 最小路径和（Minimum Path Sum）

**问题**：m×n 网格中每格有非负数，找一条从左上到右下的路径使路径上的数字总和最小。

```swift
// Swift
func minPathSum(_ grid: [[Int]]) -> Int {
    let m = grid.count, n = grid[0].count
    var dp = [Int](repeating: 0, count: n)
    dp[0] = grid[0][0]
    for j in 1..<n { dp[j] = dp[j - 1] + grid[0][j] }
    
    for i in 1..<m {
        dp[0] += grid[i][0]
        for j in 1..<n {
            dp[j] = min(dp[j], dp[j - 1]) + grid[i][j]
        }
    }
    return dp[n - 1]
}
```

```objectivec

    }
    NSInteger result = dp[n - 1];
    free(dp);
    return result;
}
```

### 7.3 三角形最小路径和（Triangle）

**问题**：自顶向下的三角形数组，求从顶到底的最小路径和。

**优化**：自底向上 DP，空间 O(n)。

```swift
// Swift - 自底向上
func minimumTotal(_ triangle: [[Int]]) -> Int {
    var dp = triangle.last!
    for i in stride(from: triangle.count - 2, through: 0, by: -1) {
        for j in 0...i {
            dp[j] = triangle[i][j] + min(dp[j], dp[j + 1])
        }
    }
    return dp[0]
}
```

```objectivec
// Objective-C
- (NSInteger)minimumTotal:(NSArray<NSArray<NSNumber *> *> *)triangle {

    NSInteger *dp = malloc(n * sizeof(NSInteger));
    NSArray<NSNumber *> *lastRow = triangle[n - 1];
    for (NSInteger j = 0; j < n; j++) dp[j] = lastRow[j].integerValue;
    
    for (NSInteger i = n - 2; i >= 0; i--) {
        for (NSInteger j = 0; j <= i; j++) {
            dp[j] = triangle[i][j].integerValue + MIN(dp[j], dp[j + 1]);
        }
    }
    NSInteger result = dp[0];

    return result;
}
```

## 八、状态机 DP（State Machine DP）

### 8.1 概念

某些 DP 问题中，每个位置的状态不是单一的，而是有多种「子状态」（如持有/不持有股票、偷/不偷、前一次是什么操作）。可以将其建模为有限状态机。

### 8.2 打家劫舍 I（House Robber）

**问题**：相邻房屋不能同时偷，求最大金额。

**状态机**：每个位置有两个状态——偷 / 不偷
- rob[i] = 偷第 i 家的最大收益 = notRob[i-1] + nums[i]
- notRob[i] = 不偷第 i 家的最大收益 = max(rob[i-1], notRob[i-1])

```swift
// Swift
func rob(_ nums: [Int]) -> Int {
    var robPrev = 0, notRobPrev = 0
    for num in nums {
        let robCurr = notRobPrev + num
        let notRobCurr = max(robPrev, notRobPrev)
        robPrev = robCurr
        notRobPrev = notRobCurr
    }
    return max(robPrev, notRobPrev)
}
```

```objectivec
// Objective-C
- (NSInteger)rob:(NSArray<NSNumber *> *)nums {
    NSInteger robPrev = 0, notRobPrev = 0;
    for (NSNumber *numObj in nums) {
        NSInteger num = numObj.integerValue;
        NSInteger robCurr = notRobPrev + num;
        NSInteger notRobCurr = MAX(robPrev, notRobPrev);
        robPrev = robCurr;
        notRobPrev = notRobCurr;
    }
    return MAX(robPrev, notRobPrev);
}
```

### 8.3 打家劫舍 II（环形）

**问题**：房屋围成环形（首尾相邻）。

**思路**：分两种情况——要么不偷第一家（对 nums[1..n-1] 做 House Robber I），要么不偷最后一家（对 nums[0..n-2] 做）。取两者最大。

```swift
// Swift
func robII(_ nums: [Int]) -> Int {
    let n = nums.count
    if n == 1 { return nums[0] }
    
    func robRange(_ start: Int, _ end: Int) -> Int {
        var robPrev = 0, notRobPrev = 0
        for i in start...end {
            let robCurr = notRobPrev + nums[i]
            let notRobCurr = max(robPrev, notRobPrev)
            robPrev = robCurr
            notRobPrev = notRobCurr
        }
        return max(robPrev, notRobPrev)
    }
    
    return max(robRange(0, n - 2), robRange(1, n - 1))
}
```

### 8.4 股票买卖系列（Best Time to Buy and Sell Stock）

股票系列是状态机 DP 的经典代表。通用框架：

**状态定义**：dp[i][k][s]
- i：第 i 天
- k：已完成的交易次数上限
- s：当前是否持有股票（0=不持有，1=持有）

**转移方程**（通用）：
```
dp[i][k][0] = max(dp[i-1][k][0], dp[i-1][k][1] + prices[i])  // 不动 or 卖出
dp[i][k][1] = max(dp[i-1][k][1], dp[i-1][k-1][0] - prices[i])  // 不动 or 买入
```

#### 情况一：至多 1 次交易（LeetCode 121）

```swift
// Swift - 一次扫描
func maxProfit(_ prices: [Int]) -> Int {
    var minPrice = Int.max
    var maxProfit = 0
    for price in prices {
        minPrice = min(minPrice, price)
        maxProfit = max(maxProfit, price - minPrice)
    }
    return maxProfit
}
```

#### 情况二：不限交易次数（LeetCode 122）

```swift
// Swift
func maxProfitII(_ prices: [Int]) -> Int {
    var profit = 0
    for i in 1..<prices.count {
        if prices[i] > prices[i - 1] {
            profit += prices[i] - prices[i - 1]
        }
    }
    return profit
}
```

#### 情况三：至多 k 次交易（LeetCode 188）

```swift
// Swift - 通用状态机
func maxProfitK(_ k: Int, _ prices: [Int]) -> Int {
    let n = prices.count
    if n < 2 { return 0 }
    
    // 当 k >= n/2 时等价于无限次交易
    if k >= n / 2 {
        var profit = 0
        for i in 1..<n {
            profit += max(0, prices[i] - prices[i - 1])
        }
        return profit
    }
    
    // dp[j][0] = 完成至多 j 次交易且不持有的最大利润
    // dp[j][1] = 完成至多 j 次交易且持有的最大利润
    var dp = [[Int]](repeating: [0, Int.min], count: k + 1)
    
    for price in prices {
        for j in stride(from: k, through: 1, by: -1) {
            dp[j][0] = max(dp[j][0], dp[j][1] + price)
            dp[j][1] = max(dp[j][1], dp[j - 1][0] - price)
        }
    }
    return dp[k][0]
}
```

```objectivec
// Objective-C - 至多 k 次
- (NSInteger)maxProfit:(NSInteger)k prices:(NSArray<NSNumber *> *)prices {
    NSInteger n = prices.count;
    if (n < 2) return 0;
    
    if (k >= n / 2) {
        NSInteger profit = 0;
        for (NSInteger i = 1; i < n; i++) {
            NSInteger diff = prices[i].integerValue - prices[i - 1].integerValue;
            if (diff > 0) profit += diff;
        }
        return profit;
    }
    
    // dp[j][0/1]
    NSInteger (*dp)[2] = calloc(k + 1, sizeof(NSInteger[2]));
    for (NSInteger j = 0; j <= k; j++) { dp[j][0] = 0; dp[j][1] = NSIntegerMin; }
    
    for (NSInteger i = 0; i < n; i++) {
        NSInteger price = prices[i].integerValue;
        for (NSInteger j = k; j >= 1; j--) {
            dp[j][0] = MAX(dp[j][0], dp[j][1] + price);
            dp[j][1] = MAX(dp[j][1], dp[j - 1][0] - price);
        }
    }
    NSInteger result = dp[k][0];
    free(dp);
    return result;
}
```

#### 情况四：含冷冻期（LeetCode 309）

卖出后需冷冻 1 天。状态：持有 / 不持有且不在冷冻 / 不持有且在冷冻期

```swift
// Swift
func maxProfitWithCooldown(_ prices: [Int]) -> Int {
    var hold = Int.min    // 持有股票
    var sold = 0          // 刚卖出（冷冻期）
    var rest = 0          // 不持有且不在冷冻
    
    for price in prices {
        let prevSold = sold
        sold = hold + price
        hold = max(hold, rest - price)
        rest = max(rest, prevSold)
    }
    return max(sold, rest)
}
```

#### 情况五：含手续费（LeetCode 714）

```swift
// Swift
func maxProfitWithFee(_ prices: [Int], _ fee: Int) -> Int {
    var hold = -prices[0]
    var cash = 0
    for i in 1..<prices.count {
        let newCash = max(cash, hold + prices[i] - fee)
        let newHold = max(hold, cash - prices[i])
        cash = newCash
        hold = newHold
    }
    return cash
}
```

## 九、树形 DP（Tree DP）

### 9.1 概念

在树结构上进行 DP，通常利用 DFS 后序遍历，从叶子往根推导。

### 9.2 打家劫舍 III（House Robber III）

**问题**：二叉树结构的房屋，相邻（父子）节点不能同时偷。

```swift
// Swift
class TreeNode {
    var val: Int
    var left: TreeNode?
    var right: TreeNode?
    init(_ val: Int) { self.val = val }
}

func rob(_ root: TreeNode?) -> Int {
    func dfs(_ node: TreeNode?) -> (rob: Int, notRob: Int) {
        guard let node = node else { return (0, 0) }
        let left = dfs(node.left)
        let right = dfs(node.right)
        // 偷当前节点：左右孩子不能偷
        let robCurr = node.val + left.notRob + right.notRob
        // 不偷当前节点：左右孩子可偷可不偷
        let notRobCurr = max(left.rob, left.notRob) + max(right.rob, right.notRob)
        return (robCurr, notRobCurr)
    }
    let result = dfs(root)
    return max(result.rob, result.notRob)
}
```

```objectivec
// Objective-C
typedef struct {
    NSInteger rob;
    NSInteger notRob;
} RobResult;

@implementation Solution

- (NSInteger)rob:(TreeNode *)root {
    RobResult result = [self dfs:root];
    return MAX(result.rob, result.notRob);
}

- (RobResult)dfs:(TreeNode *)node {
    if (!node) return (RobResult){0, 0};
    RobResult left = [self dfs:node.left];
    RobResult right = [self dfs:node.right];
    NSInteger robCurr = node.val + left.notRob + right.notRob;
    NSInteger notRobCurr = MAX(left.rob, left.notRob) + MAX(right.rob, right.notRob);
    return (RobResult){robCurr, notRobCurr};
}

@end
```

**复杂度**：时间 O(n)，空间 O(h)（h 为树高，栈空间）

### 9.3 二叉树直径（Diameter of Binary Tree）

```swift
// Swift
func diameterOfBinaryTree(_ root: TreeNode?) -> Int {
    var maxDiameter = 0
    
    @discardableResult
    func depth(_ node: TreeNode?) -> Int {
        guard let node = node else { return 0 }
        let leftDepth = depth(node.left)
        let rightDepth = depth(node.right)
        maxDiameter = max(maxDiameter, leftDepth + rightDepth)
        return 1 + max(leftDepth, rightDepth)
    }
    
    depth(root)
    return maxDiameter
}
```

## 十、状态压缩 DP（Bitmask DP）

### 10.1 概念

当状态需要记录「哪些元素已被使用/访问」时，可用二进制位掩码（bitmask）表示集合状态。适用于元素个数 ≤ 20（2^20 ≈ 100 万）。

### 10.2 旅行商问题（TSP）

**问题**：从城市 0 出发，经过所有城市恰好一次后回到起点，求最小路径权重。

**状态**：dp[mask][i] = 已访问集合为 mask、当前在城市 i 的最小代价
**转移**：dp[mask | (1<<j)][j] = min(dp[mask][i] + dist[i][j]) for all j not in mask

```swift
// Swift - TSP with Bitmask DP
func tsp(_ dist: [[Int]]) -> Int {
    let n = dist.count
    let full = (1 << n) - 1
    var dp = [[Int]](repeating: [Int](repeating: Int.max / 2, count: n), count: 1 << n)
    dp[1][0] = 0  // 从城市 0 出发，已访问 {0}
    
    for mask in 1..<(1 << n) {
        for i in 0..<n {
            if mask & (1 << i) == 0 { continue }
            if dp[mask][i] == Int.max / 2 { continue }
            for j in 0..<n {
                if mask & (1 << j) != 0 { continue }
                let nextMask = mask | (1 << j)
                dp[nextMask][j] = min(dp[nextMask][j], dp[mask][i] + dist[i][j])
            }
        }
    }
    
    // 所有城市访问完，回到 0
    var result = Int.max
    for i in 1..<n {
        result = min(result, dp[full][i] + dist[i][0])
    }
    return result
}
```

```objectivec
// Objective-C
- (NSInteger)tsp:(NSArray<NSArray<NSNumber *> *> *)dist {
    NSInteger n = dist.count;
    NSInteger total = 1 << n;
    NSInteger full = total - 1;
    
    NSInteger **dp = malloc(total * sizeof(NSInteger *));
    for (NSInteger s = 0; s < total; s++) {
        dp[s] = malloc(n * sizeof(NSInteger));
        for (NSInteger i = 0; i < n; i++) dp[s][i] = NSIntegerMax / 2;
    }
    dp[1][0] = 0;
    
    for (NSInteger mask = 1; mask < total; mask++) {
        for (NSInteger i = 0; i < n; i++) {
            if (!(mask & (1 << i))) continue;
            if (dp[mask][i] == NSIntegerMax / 2) continue;
            for (NSInteger j = 0; j < n; j++) {
                if (mask & (1 << j)) continue;
                NSInteger nextMask = mask | (1 << j);
                NSInteger cost = dp[mask][i] + dist[i][j].integerValue;
                dp[nextMask][j] = MIN(dp[nextMask][j], cost);
            }
        }
    }
    
    NSInteger result = NSIntegerMax;
    for (NSInteger i = 1; i < n; i++) {
        result = MIN(result, dp[full][i] + dist[i][0].integerValue);
    }
    
    for (NSInteger s = 0; s < total; s++) free(dp[s]);
    free(dp);
    return result;
}
```

**复杂度**：时间 O(2^n × n²)，空间 O(2^n × n)

### 10.3 分配问题（Assignment Problem）

**问题**：n 个人分配 n 个任务，每人恰好做一个任务，cost[i][j] 为第 i 人做第 j 任务的代价，求最小总代价。

```swift
// Swift
func minCostAssignment(_ cost: [[Int]]) -> Int {
    let n = cost.count
    var dp = [Int](repeating: Int.max / 2, count: 1 << n)
    dp[0] = 0
    
    for mask in 0..<(1 << n) {
        let person = mask.nonzeroBitCount  // 当前要分配第几个人
        if person >= n { continue }
        for task in 0..<n {
            if mask & (1 << task) != 0 { continue }
            let nextMask = mask | (1 << task)
            dp[nextMask] = min(dp[nextMask], dp[mask] + cost[person][task])
        }
    }
    return dp[(1 << n) - 1]
}
```

## 十一、数位 DP（Digit DP）

### 11.1 概念

数位 DP 用于解决「在 [L, R] 范围内满足某种性质的整数有多少个」的计数问题。核心思想是逐位决策，维护「是否仍被上界约束（tight）」的状态。

### 11.2 通用框架

```swift
// Swift - 数位 DP 通用模板
func countDigitDP(_ n: Int) -> Int {
    let digits = String(n).map { Int(String($0))! }
    let len = digits.count
    // memo[pos][tight][...其他状态]
    var memo = [[[Int?]]](repeating: [[Int?]](repeating: [Int?](repeating: nil, count: /* 状态 */), count: 2), count: len)
    
    func dfs(_ pos: Int, _ tight: Bool, _ state: Int) -> Int {
        if pos == len { return /* 判断是否满足条件 */ 1 }
        if let cached = memo[pos][tight ? 1 : 0][state] { return cached }
        
        let limit = tight ? digits[pos] : 9
        var count = 0
        for d in 0...limit {
            let newTight = tight && (d == limit)
            let newState = /* 更新状态 */ state
            count += dfs(pos + 1, newTight, newState)
        }
        memo[pos][tight ? 1 : 0][state] = count
        return count
    }
    
    return dfs(0, true, /* 初始状态 */ 0)
}
```

### 11.3 示例：统计各位数字都不同的数（LeetCode 357）

**问题**：给定 n，返回 [0, 10^n) 中各位数字都不同的数的个数。

```swift
// Swift - 组合数学解法
func countNumbersWithUniqueDigits(_ n: Int) -> Int {
    if n == 0 { return 1 }
    var result = 10  // n=1 时有 10 个
    var uniqueDigits = 9  // 首位有 9 种选择（1-9）
    var availableNumber = 9  // 后续可选数字
    
    for _ in 2...min(n, 10) {
        uniqueDigits *= availableNumber
        result += uniqueDigits
        availableNumber -= 1
    }
    return result
}
```

```swift
// Swift - 数位 DP 解法（更通用）
func countNumbersWithUniqueDigitsDP(_ n: Int) -> Int {
    if n == 0 { return 1 }
    let upper = Int(pow(10.0, Double(n))) - 1
    let digits = String(upper).map { Int(String($0))! }
    let len = digits.count
    
    // memo[pos][tight][usedMask] - usedMask 记录已使用的数字
    var memo: [String: Int] = [:]
    
    func dfs(_ pos: Int, _ tight: Bool, _ usedMask: Int, _ started: Bool) -> Int {
        if pos == len { return 1 }
        let key = "\(pos)_\(tight)_\(usedMask)_\(started)"
        if let cached = memo[key] { return cached }
        
        let limit = tight ? digits[pos] : 9
        var count = 0
        for d in 0...limit {
            if started && (usedMask & (1 << d)) != 0 { continue }
            let newTight = tight && (d == limit)
            let newStarted = started || (d != 0)
            let newMask = newStarted ? (usedMask | (1 << d)) : 0
            count += dfs(pos + 1, newTight, newMask, newStarted)
        }
        memo[key] = count
        return count
    }
    
    return dfs(0, true, 0, false)
}
```

## 十二、概率/期望 DP

### 12.1 概念

当问题涉及随机过程时，DP 的值可以是概率或期望值。状态转移变为加权求和（权重为转移概率）。

### 12.2 骑士拨号器（Knight Dialer, LeetCode 935）

**问题**：国际象棋的骑士在电话键盘上跳 n-1 步，能拨出多少种不同的 n 位号码？

```swift
// Swift
func knightDialer(_ n: Int) -> Int {
    let MOD = 1_000_000_007
    // 从数字 i 可以跳到的数字集合
    let jumps: [[Int]] = [
        [4, 6],     // 0 -> 4, 6
        [6, 8],     // 1 -> 6, 8
        [7, 9],     // 2 -> 7, 9
        [4, 8],     // 3 -> 4, 8
        [0, 3, 9],  // 4 -> 0, 3, 9
        [],         // 5 -> 无
        [0, 1, 7],  // 6 -> 0, 1, 7
        [2, 6],     // 7 -> 2, 6
        [1, 3],     // 8 -> 1, 3
        [2, 4]      // 9 -> 2, 4
    ]
    
    var dp = [Int](repeating: 1, count: 10)  // 长度 1 时每个起点 1 种
    
    for _ in 1..<n {
        var newDp = [Int](repeating: 0, count: 10)
        for digit in 0...9 {
            for prev in jumps[digit] {
                newDp[digit] = (newDp[digit] + dp[prev]) % MOD
            }
        }
        dp = newDp
    }
    
    return dp.reduce(0) { ($0 + $1) % MOD }
}
```

```objectivec
// Objective-C
- (NSInteger)knightDialer:(NSInteger)n {
    NSInteger MOD = 1000000007;
    NSArray<NSArray<NSNumber *> *> *jumps = @[
        @[@4, @6], @[@6, @8], @[@7, @9], @[@4, @8],
        @[@0, @3, @9], @[], @[@0, @1, @7],
        @[@2, @6], @[@1, @3], @[@2, @4]
    ];
    
    NSInteger dp[10], newDp[10];
    for (int i = 0; i < 10; i++) dp[i] = 1;
    
    for (NSInteger step = 1; step < n; step++) {
        memset(newDp, 0, sizeof(newDp));
        for (int digit = 0; digit < 10; digit++) {
            for (NSNumber *prevObj in jumps[digit]) {
                newDp[digit] = (newDp[digit] + dp[prevObj.intValue]) % MOD;
            }
        }
        memcpy(dp, newDp, sizeof(dp));
    }
    
    NSInteger result = 0;
    for (int i = 0; i < 10; i++) result = (result + dp[i]) % MOD;
    return result;
}
```

## 十三、经典 DP 综合问题

### 13.1 单词拆分（Word Break, LeetCode 139）

**问题**：判断字符串 s 是否可以被空格拆分为字典中的单词。

**状态**：dp[i] = s[0..i-1] 是否可以被拆分
**转移**：dp[i] = any(dp[j] && s[j..i-1] in dict) for j in [0, i-1]

```swift
// Swift
func wordBreak(_ s: String, _ wordDict: [String]) -> Bool {
    let chars = Array(s)
    let n = chars.count
    let wordSet = Set(wordDict)
    var dp = [Bool](repeating: false, count: n + 1)
    dp[0] = true
    
    for i in 1...n {
        for j in 0..<i {
            if dp[j] && wordSet.contains(String(chars[j..<i])) {
                dp[i] = true
                break
            }
        }
    }
    return dp[n]
}
```

```objectivec
// Objective-C
- (BOOL)wordBreak:(NSString *)s wordDict:(NSArray<NSString *> *)wordDict {
    NSSet<NSString *> *wordSet = [NSSet setWithArray:wordDict];
    NSInteger n = s.length;
    BOOL *dp = calloc(n + 1, sizeof(BOOL));
    dp[0] = YES;
    
    for (NSInteger i = 1; i <= n; i++) {
        for (NSInteger j = 0; j < i; j++) {
            if (dp[j]) {
                NSString *sub = [s substringWithRange:NSMakeRange(j, i - j)];
                if ([wordSet containsObject:sub]) {
                    dp[i] = YES;
                    break;
                }
            }
        }
    }
    BOOL result = dp[n];
    free(dp);
    return result;
}
```

**复杂度**：时间 O(n² × k)（k 为子串比较），可优化为 O(n × maxWordLen)

### 13.2 解码方法（Decode Ways, LeetCode 91）

**问题**：数字字符串，'1'->'A', '2'->'B', ..., '26'->'Z'，求解码方法数。

```swift
// Swift
func numDecodings(_ s: String) -> Int {
    let chars = Array(s)
    let n = chars.count
    if n == 0 || chars[0] == "0" { return 0 }
    
    var prev2 = 1  // dp[i-2]
    var prev1 = 1  // dp[i-1]
    
    for i in 1..<n {
        var curr = 0
        let oneDigit = Int(String(chars[i]))!
        let twoDigit = Int(String(chars[i-1...i]))!
        
        if oneDigit >= 1 { curr += prev1 }
        if twoDigit >= 10 && twoDigit <= 26 { curr += prev2 }
        
        prev2 = prev1
        prev1 = curr
    }
    return prev1
}
```

### 13.3 最长有效括号（Longest Valid Parentheses, LeetCode 32）

**状态**：dp[i] = 以 s[i] 结尾的最长有效括号长度
**转移**：
- 若 s[i] == '('：dp[i] = 0
- 若 s[i] == ')' 且 s[i-1] == '('：dp[i] = dp[i-2] + 2
- 若 s[i] == ')' 且 s[i-1] == ')' 且 s[i-1-dp[i-1]] == '('：dp[i] = dp[i-1] + 2 + dp[i-2-dp[i-1]]

```swift
// Swift
func longestValidParentheses(_ s: String) -> Int {
    let chars = Array(s)
    let n = chars.count
    if n < 2 { return 0 }
    
    var dp = [Int](repeating: 0, count: n)
    var maxLen = 0
    
    for i in 1..<n {
        if chars[i] == ")" {
            if chars[i - 1] == "(" {
                dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2
            } else if dp[i - 1] > 0 {
                let matchIdx = i - 1 - dp[i - 1]
                if matchIdx >= 0 && chars[matchIdx] == "(" {
                    dp[i] = dp[i - 1] + 2 + (matchIdx >= 1 ? dp[matchIdx - 1] : 0)
                }
            }
            maxLen = max(maxLen, dp[i])
        }
    }
    return maxLen
}
```

### 13.4 正则表达式匹配（Regular Expression Matching, LeetCode 10）

**问题**：实现支持 '.' 和 '*' 的正则匹配。

**状态**：dp[i][j] = s[0..i-1] 是否匹配 p[0..j-1]
**转移**：
- 若 p[j-1] == '.' 或 p[j-1] == s[i-1]：dp[i][j] = dp[i-1][j-1]
- 若 p[j-1] == '*'：
  - 匹配 0 次：dp[i][j] = dp[i][j-2]
  - 匹配 1+ 次（前提 p[j-2] matches s[i-1]）：dp[i][j] = dp[i-1][j]

```swift
// Swift
func isMatch(_ s: String, _ p: String) -> Bool {
    let sc = Array(s), pc = Array(p)
    let m = sc.count, n = pc.count
    var dp = [[Bool]](repeating: [Bool](repeating: false, count: n + 1), count: m + 1)
    dp[0][0] = true
    
    // 初始化：空字符串匹配 x*y*z* 模式
    for j in 1...n {
        if pc[j - 1] == "*" {
            dp[0][j] = dp[0][j - 2]
        }
    }
    
    for i in 1...m {
        for j in 1...n {
            if pc[j - 1] == "." || pc[j - 1] == sc[i - 1] {
                dp[i][j] = dp[i - 1][j - 1]
            } else if pc[j - 1] == "*" {
                // * 匹配 0 次前面的字符
                dp[i][j] = dp[i][j - 2]
                // * 匹配 1+ 次（前面字符能匹配当前 s 字符）
                if pc[j - 2] == "." || pc[j - 2] == sc[i - 1] {
                    dp[i][j] = dp[i][j] || dp[i - 1][j]
                }
            }
        }
    }
    return dp[m][n]
}
```

```objectivec
// Objective-C
- (BOOL)isMatch:(NSString *)s pattern:(NSString *)p {
    NSInteger m = s.length, n = p.length;
    BOOL **dp = calloc(m + 1, sizeof(BOOL *));
    for (NSInteger i = 0; i <= m; i++) dp[i] = calloc(n + 1, sizeof(BOOL));
    dp[0][0] = YES;
    
    for (NSInteger j = 1; j <= n; j++) {
        if ([p characterAtIndex:j - 1] == '*') {
            dp[0][j] = dp[0][j - 2];
        }
    }
    
    for (NSInteger i = 1; i <= m; i++) {
        unichar si = [s characterAtIndex:i - 1];
        for (NSInteger j = 1; j <= n; j++) {
            unichar pj = [p characterAtIndex:j - 1];
            if (pj == '.' || pj == si) {
                dp[i][j] = dp[i - 1][j - 1];
            } else if (pj == '*') {
                dp[i][j] = dp[i][j - 2];
                unichar pjPrev = [p characterAtIndex:j - 2];
                if (pjPrev == '.' || pjPrev == si) {
                    dp[i][j] = dp[i][j] || dp[i - 1][j];
                }
            }
        }
    }
    
    BOOL result = dp[m][n];
    for (NSInteger i = 0; i <= m; i++) free(dp[i]);
    free(dp);
    return result;
}
```

**复杂度**：时间 O(m×n)，空间 O(m×n)

### 13.5 通配符匹配（Wildcard Matching, LeetCode 44）

**问题**：'?' 匹配单个字符，'*' 匹配任意序列（含空）。

```swift
// Swift
func isMatchWildcard(_ s: String, _ p: String) -> Bool {
    let sc = Array(s), pc = Array(p)
    let m = sc.count, n = pc.count
    
    // 空间优化为一行
    var dp = [Bool](repeating: false, count: n + 1)
    dp[0] = true
    for j in 1...n where pc[j - 1] == "*" {
        dp[j] = dp[j - 1]
    }
    
    for i in 1...m {
        var newDp = [Bool](repeating: false, count: n + 1)
        for j in 1...n {
            if pc[j - 1] == "?" || pc[j - 1] == sc[i - 1] {
                newDp[j] = dp[j - 1]  // 上一行左上角
            } else if pc[j - 1] == "*" {
                newDp[j] = dp[j] || newDp[j - 1]
                // dp[j] = * 匹配至少一个（上方，即 i-1 行同 j 列）
                // newDp[j-1] = * 匹配空（左方）
            }
        }
        dp = newDp
    }
    return dp[n]
}
```

## 十四、DP 优化技巧

### 14.1 空间优化总结

| 原始复杂度 | 优化方式 | 优化后 | 适用条件 |
|-----------|---------|--------|---------|
| O(n²) | 滚动数组（2行） | O(n) | dp[i] 只依赖 dp[i-1] |
| O(n) | 滚动变量 | O(1) | dp[i] 只依赖常数个前驱 |
| O(n×m) | 一维+prev变量 | O(m) | dp[i][j] 依赖 dp[i-1][j-1], dp[i-1][j], dp[i][j-1] |

### 14.2 单调队列/单调栈优化

当转移方程形如 dp[i] = min/max(dp[j] + cost) for j in [i-k, i-1] 时（滑动窗口），可用单调队列将每次转移从 O(k) 优化到均摊 O(1)。

**示例：滑动窗口最大值辅助 DP**

```swift
// Swift - 单调队列优化（约束长度的最小代价问题）
func minCostWithWindow(_ costs: [Int], _ k: Int) -> Int {
    let n = costs.count
    var dp = [Int](repeating: Int.max, count: n + 1)
    dp[0] = 0
    var deque: [Int] = [0]  // 存储下标，对应 dp 值单调递增
    
    for i in 1...n {
        // 移除超出窗口的元素
        while !deque.isEmpty && deque.first! < i - k {
            deque.removeFirst()
        }
        dp[i] = dp[deque.first!] + costs[i - 1]
        // 维护单调性
        while !deque.isEmpty && dp[i] <= dp[deque.last!] {
            deque.removeLast()
        }
        deque.append(i)
    }
    return dp[n]
}
```

### 14.3 矩阵快速幂优化

当线性递推关系固定时（如 dp[i] = a×dp[i-1] + b×dp[i-2]），可用矩阵快速幂将 O(n) 优化到 O(k³ log n)，其中 k 为状态数。

**示例：Fibonacci 的 O(log n) 求法**

```swift
// Swift - 矩阵快速幂
typealias Matrix = [[Int]]

func multiply(_ a: Matrix, _ b: Matrix, _ mod: Int) -> Matrix {
    let n = a.count
    var c = Matrix(repeating: [Int](repeating: 0, count: n), count: n)
    for i in 0..<n {
        for j in 0..<n {
            for k in 0..<n {
                c[i][j] = (c[i][j] + a[i][k] * b[k][j]) % mod
            }
        }
    }
    return c
}

func matPow(_ mat: Matrix, _ p: Int, _ mod: Int) -> Matrix {
    let n = mat.count
    var result = Matrix(repeating: [Int](repeating: 0, count: n), count: n)
    for i in 0..<n { result[i][i] = 1 }  // 单位矩阵
    var base = mat
    var exp = p
    while exp > 0 {
        if exp % 2 == 1 { result = multiply(result, base, mod) }
        base = multiply(base, base, mod)
        exp /= 2
    }
    return result
}

func fibFast(_ n: Int) -> Int {
    if n <= 1 { return n }
    let mod = 1_000_000_007
    let mat: Matrix = [[1, 1], [1, 0]]
    let result = matPow(mat, n - 1, mod)
    return result[0][0]  // fib(n)
}
```

### 14.4 斜率优化（Convex Hull Trick）

当转移方程形如 dp[i] = min(dp[j] + f(i)×g(j) + h(i)) 且 g(j) 单调时，可以用凸包维护直线集合，将每次转移从 O(n) 优化到 O(1) 或 O(log n)。

这是竞赛级优化，面试中较少涉及，但理解思想有助于处理特殊形式的 DP。

### 16.2 iOS 中的 DP 实战：文本自动断行

```swift
// Swift - 简化的 Knuth-Plass 断行算法
func optimalLineBreaks(_ words: [String], lineWidth: Int) -> [[String]] {
    let n = words.count
    let wordLens = words.map { $0.count }
    
    // dp[i] = 从第 i 个词到末尾的最小代价
    var dp = [Int](repeating: Int.max / 2, count: n + 1)
    var breaks = [Int](repeating: 0, count: n + 1)
    dp[n] = 0
    
    for i in stride(from: n - 1, through: 0, by: -1) {
        var lineLen = 0
        for j in i..<n {
            lineLen += wordLens[j] + (j > i ? 1 : 0)  // 空格
            if lineLen > lineWidth { break }
            
            let slack = lineWidth - lineLen
            let cost = (j == n - 1) ? 0 : slack * slack  // 最后一行不惩罚
            if cost + dp[j + 1] < dp[i] {
                dp[i] = cost + dp[j + 1]
                breaks[i] = j + 1
            }
        }
    }
    
    // 根据 breaks 数组重建分行
    var lines: [[String]] = []
    var i = 0
    while i < n {
        lines.append(Array(words[i..<breaks[i]]))
        i = breaks[i]
    }
    return lines
}
```

```swift
// Swift - 多列瀑布流高度均衡分配（DP 解法）
func assignItemsToColumns(_ heights: [CGFloat], columns: Int) -> [[Int]] {
    let n = heights.count
    // 贪心解法：每次将下一个 item 放入当前最矮的列
    var columnHeights = [CGFloat](repeating: 0, count: columns)
    var assignment = [[Int]](repeating: [], count: columns)
    
    for i in 0..<n {
        // 找到最矮的列
        var minCol = 0
        for c in 1..<columns {
            if columnHeights[c] < columnHeights[minCol] {
                minCol = c
            }
        }
        assignment[minCol].append(i)
        columnHeights[minCol] += heights[i]
    }
    return assignment
}
```

### 17.2 完全平方数（Perfect Squares, LeetCode 279）

**问题**：找到若干完全平方数使其和等于 n，求最少个数。

```swift
// Swift - 完全背包思路
func numSquares(_ n: Int) -> Int {
    var dp = [Int](repeating: Int.max, count: n + 1)
    dp[0] = 0
    
    for i in 1...n {
        var j = 1
        while j * j <= i {
            dp[i] = min(dp[i], dp[i - j * j] + 1)
            j += 1
        }
    }
    return dp[n]
}
```

```objectivec
// Objective-C
- (NSInteger)numSquares:(NSInteger)n {
    NSInteger *dp = malloc((n + 1) * sizeof(NSInteger));

    dp[0] = 0;

    }
    NSInteger result = dp[n];
    free(dp);
    return result;

```

**复杂度**：时间 O(n√n)，空间 O(n)

### 17.3 最大正方形（Maximal Square, LeetCode 221）

**问题**：在只含 '0' 和 '1' 的 m×n 矩阵中，找到只包含 1 的最大正方形面积。

**状态**：dp[i][j] = 以 (i,j) 为右下角的最大全 1 正方形边长
**转移**：dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1（前提 matrix[i][j] == '1'）

```swift
// Swift
func maximalSquare(_ matrix: [[Character]]) -> Int {
    let m = matrix.count, n = matrix[0].count
    var dp = [Int](repeating: 0, count: n + 1)
    var maxSide = 0, prev = 0
    
    for i in 1...m {
        for j in 1...n {
            let temp = dp[j]
            if matrix[i - 1][j - 1] == "1" {
                dp[j] = min(dp[j], dp[j - 1], prev) + 1
                maxSide = max(maxSide, dp[j])
            } else {
                dp[j] = 0
            }
            prev = temp
        }
        prev = 0
    }
    return maxSide * maxSide
}
```

```objectivec
// Objective-C
- (NSInteger)maximalSquare:(NSArray<NSArray<NSString *> *> *)matrix {

    NSInteger *dp = calloc(n + 1, sizeof(NSInteger));

            NSInteger temp = dp[j];

            } else {
                dp[j] = 0;
            }
            prev = temp;
        }
    }
    free(dp);
    return maxSide * maxSide;
}
```

### 17.4 乘积最大子数组（Maximum Product Subarray, LeetCode 152）

**关键**：负数×负数可能变最大，所以同时维护最大值和最小值。

```swift
// Swift
func maxProduct(_ nums: [Int]) -> Int {
    var maxProd = nums[0]
    var minProd = nums[0]
    var result = nums[0]
    
    for i in 1..<nums.count {
        let candidates = [nums[i], maxProd * nums[i], minProd * nums[i]]
        maxProd = candidates.max()!
        minProd = candidates.min()!
        result = max(result, maxProd)
    }
    return result
}
```

### 17.5 不同的二叉搜索树（Unique BSTs, LeetCode 96）

**问题**：给定 n，求结构上不同的 BST 数目（卡特兰数）。

**状态**：dp[n] = n 个节点能组成的不同 BST 数目
**转移**：dp[n] = Σ dp[i-1] × dp[n-i] for i in [1, n]

```swift
// Swift
func numTrees(_ n: Int) -> Int {
    var dp = [Int](repeating: 0, count: n + 1)
    dp[0] = 1
    dp[1] = 1
    
    for i in 2...n {
        for j in 1...i {
            dp[i] += dp[j - 1] * dp[i - j]
        }
    }
    return dp[n]
}
```

**数学公式**：C(n) = C(2n, n) / (n+1)（第 n 个卡特兰数）

### 17.6 交错字符串（Interleaving String, LeetCode 97）

**问题**：判断 s3 是否由 s1 和 s2 交错组成。

```swift
// Swift
func isInterleave(_ s1: String, _ s2: String, _ s3: String) -> Bool {
    let c1 = Array(s1), c2 = Array(s2), c3 = Array(s3)
    let m = c1.count, n = c2.count
    if m + n != c3.count { return false }
    
    var dp = [Bool](repeating: false, count: n + 1)
    
    for i in 0...m {
        for j in 0...n {
            if i == 0 && j == 0 {
                dp[j] = true
            } else if i == 0 {
                dp[j] = dp[j - 1] && c2[j - 1] == c3[j - 1]
            } else if j == 0 {
                dp[j] = dp[j] && c1[i - 1] == c3[i - 1]
            } else {
                dp[j] = (dp[j] && c1[i - 1] == c3[i + j - 1]) ||
                         (dp[j - 1] && c2[j - 1] == c3[i + j - 1])
            }
        }
    }
    return dp[n]
}
```

```swift
func countSubstrings(_ s: String) -> Int {

    var count = 0
    
    for center in 0..<(2 * n - 1) {
        var left = center / 2
        var right = left + center % 2
        while left >= 0 && right < n && chars[left] == chars[right] {
            count += 1
            left -= 1
            right += 1
        }
    }
    return count
```

### 17.8 最长回文子序列（Longest Palindromic Subsequence, LeetCode 516）

**状态**：dp[i][j] = s[i..j] 中最长回文子序列长度
**转移**：
- 若 s[i] == s[j]：dp[i][j] = dp[i+1][j-1] + 2
- 否则：dp[i][j] = max(dp[i+1][j], dp[i][j-1])

```swift
// Swift
func longestPalindromeSubseq(_ s: String) -> Int {
    let chars = Array(s)
    let n = chars.count
    var dp = [Int](repeating: 1, count: n)  // 空间优化为一维
    
    for i in stride(from: n - 2, through: 0, by: -1) {
        var prev = 0  // dp[i+1][j-1]
        for j in (i + 1)..<n {
            let temp = dp[j]
            if chars[i] == chars[j] {
                dp[j] = prev + 2
            } else {
                dp[j] = max(dp[j], dp[j - 1])
            }
            prev = temp
        }
    }
    return dp[n - 1]
}
```


## 十八、易错点与最佳实践

### 18.1 常见错误

1. **初始化遗漏**：忘记设置 dp[0] 或边界值，导致后续推导全错
2. **遍历顺序错误**：0/1 背包用正序遍历容量 → 物品被重复选取（变成完全背包）
3. **状态定义不清**：dp[i] 到底是「前 i 个」还是「以第 i 个结尾」？两者转移方程不同
4. **整数溢出**：DP 值可能很大，注意取模或使用大数
5. **空间优化后丢失信息**：滚动数组时覆盖了后续还需要的值
6. **忘记取 max/min 的初始值**：求最小值时 dp 应初始为 INF，求最大值时初始为 -INF 或 0

### 18.2 DP 最佳实践

1. **先写暴力递归**：确认逻辑正确，再转化为 DP
2. **画 DP 表格**：手动填表验证转移方程
3. **从小规模开始**：n=1,2,3 时手动验证
4. **注意「恰好」vs「至多」**：
   - "恰好装满背包" → dp[0]=0, dp[1..W]=INF
   - "不超过容量" → dp[0..W]=0
5. **路径重构**：若需要输出具体方案，额外记录每步的选择（choice 数组）
6. **计数类问题**：注意是否需要取模，以及加法溢出

### 18.3 DP vs 贪心的判断

| 条件 | 用贪心 | 用 DP |
|------|--------|-------|
| 每步的局部最优等于全局最优 | ✓ | |
| 需要考虑未来的影响 | | ✓ |
| 存在反例使贪心失败 | | ✓ |
| 有明确的贪心选择证明 | ✓ | |
| 问题有重叠子问题 | | ✓ |

**示例**：零钱兑换 [1,3,4]，凑 6
- 贪心：4+1+1 = 3枚 ✗
- DP：3+3 = 2枚 ✓

### 19.1 案例：背包问题的实际应用——任务调度最大收益

**场景**：有 n 个任务，每个任务有开始时间、结束时间、收益。同一时间只能做一个任务，求最大收益。（即加权活动选择问题 / Weighted Job Scheduling）

```swift
// Swift - 加权任务调度 (DP + 二分查找)
struct Job {
    let start: Int
    let end: Int
    let profit: Int
}

func jobScheduling(_ startTime: [Int], _ endTime: [Int], _ profit: [Int]) -> Int {
    let n = startTime.count
    var jobs = (0..<n).map { Job(start: startTime[$0], end: endTime[$0], profit: profit[$0]) }
    // 按结束时间排序
    jobs.sort { $0.end < $1.end }
    
    // dp[i] = 考虑前 i 个任务的最大收益
    var dp = [Int](repeating: 0, count: n + 1)
    
    for i in 1...n {
        let job = jobs[i - 1]
        // 二分查找最后一个结束时间 <= job.start 的任务
        var lo = 0, hi = i - 1
        while lo < hi {
            let mid = (lo + hi + 1) / 2
            if jobs[mid - 1].end <= job.start {
                lo = mid
            } else {
                hi = mid - 1
            }
        }
        // lo 是最后一个不冲突的任务索引 (1-indexed in dp)
        let include = dp[lo] + job.profit
        let exclude = dp[i - 1]
        dp[i] = max(include, exclude)
    }
    return dp[n]
}
```

```objectivec
// Objective-C
typedef struct {
    NSInteger start, end, profit;
} OCJob;

- (NSInteger)jobScheduling:(NSArray<NSNumber *> *)startTime
                   endTime:(NSArray<NSNumber *> *)endTime
                    profit:(NSArray<NSNumber *> *)profit {
    NSInteger n = startTime.count;
    OCJob *jobs = malloc(n * sizeof(OCJob));
    for (NSInteger i = 0; i < n; i++) {
        jobs[i] = (OCJob){startTime[i].integerValue, endTime[i].integerValue, profit[i].integerValue};
    }
    // 按结束时间排序
    qsort_b(jobs, n, sizeof(OCJob), ^int(const void *a, const void *b) {
        return (int)(((OCJob *)a)->end - ((OCJob *)b)->end);
    });
    
    NSInteger *dp = calloc(n + 1, sizeof(NSInteger));
    for (NSInteger i = 1; i <= n; i++) {
        OCJob job = jobs[i - 1];
        // 二分查找
        NSInteger lo = 0, hi = i - 1;
        while (lo < hi) {
            NSInteger mid = (lo + hi + 1) / 2;
            if (jobs[mid - 1].end <= job.start) lo = mid;
            else hi = mid - 1;
        }
        NSInteger include = dp[lo] + job.profit;
        NSInteger exclude = dp[i - 1];
        dp[i] = MAX(include, exclude);
    }
    NSInteger result = dp[n];
    free(jobs);
    free(dp);
    return result;
}
```

**复杂度**：排序 O(n log n) + DP O(n log n)（每次二分），总 O(n log n)

### 19.2 案例：子序列问题综合——俄罗斯套娃信封（LeetCode 354）

**问题**：给定信封 (w, h)，信封 A 能套在 B 里当且仅当 A.w < B.w && A.h < B.h。求最多能套多少个。

**思路**：按宽度升序、宽度相同时按高度降序排序，然后对高度数组求 LIS。

```swift
// Swift
func maxEnvelopes(_ envelopes: [[Int]]) -> Int {
    // 按 width 升序，width 相同按 height 降序
    let sorted = envelopes.sorted { 
        $0[0] != $1[0] ? $0[0] < $1[0] : $0[1] > $1[1] 
    }
    
    // 对 height 求 LIS（O(n log n)）
    var tails: [Int] = []
    for env in sorted {
        let h = env[1]
        var lo = 0, hi = tails.count
        while lo < hi {
            let mid = (lo + hi) / 2
            if tails[mid] < h { lo = mid + 1 }
            else { hi = mid }
        }
        if lo == tails.count { tails.append(h) }
        else { tails[lo] = h }
    }
    return tails.count
}
```

**复杂度**：O(n log n)

## 二十一、面试题集（附答案要点）

### Q1：什么是动态规划？与分治法有什么区别？

**答**：动态规划是将问题分解为重叠子问题，通过存储子问题解避免重复计算的方法。与分治法的核心区别是子问题是否重叠——分治的子问题互不重叠（如归并排序左右两半），而 DP 的子问题大量重复（如 Fibonacci 中 fib(3) 被计算多次）。

### Q2：如何判断一个问题能用 DP 解决？

**答**：检查三个性质——最优子结构（大问题的最优解包含子问题的最优解）、重叠子问题（递归过程中同一子问题被多次求解）、无后效性（状态一旦确定，未来决策不影响已确定的状态）。

### Q3：Top-Down 和 Bottom-Up 哪个更好？

**答**：各有优势。Top-Down（记忆化递归）更直观、只计算需要的子问题（稀疏子问题时更优）、易于处理复杂状态转移；Bottom-Up（迭代 DP）无递归开销、空间优化更容易、缓存友好。面试中建议先用 Top-Down 写出正确解，再优化为 Bottom-Up。

### Q4：0/1 背包与完全背包的空间优化有什么区别？为什么遍历顺序不同？

**答**：0/1 背包每个物品只能用一次，内层逆序遍历保证 `dp[w-wt[i]]` 是上一行（未选当前物品）的值；完全背包物品可重复使用，内层正序遍历使 `dp[w-wt[i]]` 是当前行的值（已考虑选了当前物品），允许重复选取。

### Q5：什么是状态压缩 DP？举例说明。

**答**：当状态需要记录一个集合中哪些元素被选择时，用二进制位表示集合状态。典型例子是 TSP（旅行商问题）：`dp[mask][i]` 表示已访问城市集合为 mask、当前在城市 i 的最短路径。mask 是 n 位二进制数，第 k 位为 1 表示城市 k 已访问。时间复杂度 O(2^n * n^2)。

### Q6：编辑距离问题的状态转移方程是什么？

**答**：`dp[i][j]` 表示 word1 前 i 个字符变为 word2 前 j 个字符的最小操作数。若 `word1[i-1] == word2[j-1]`，则 `dp[i][j] = dp[i-1][j-1]`；否则 `dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])`，分别对应删除、插入、替换。

### Q7：最长递增子序列有哪些解法？各自复杂度？

**答**：(1) 朴素 DP O(n²)：`dp[i]` 为以 nums[i] 结尾的 LIS 长度；(2) 贪心+二分 O(n log n)：维护 tails 数组，tails[k] 是长度为 k+1 的递增子序列的最小末尾，用二分查找更新；(3) 树状数组/线段树优化 O(n log n)：离散化后用数据结构加速前缀最大值查询。

### Q8：如何输出 DP 的具体方案而不只是最优值？

**答**：方法一：记录 choice/parent 数组，回溯路径；方法二：从最终状态逆向检查状态转移条件。例如 LCS 中，从 dp[m][n] 开始，若两字符相等则加入结果并左上移动，否则向 dp 值更大的方向移动。

## 二十二、易错点与最佳实践

### 常见错误

1. **初始化遗漏**：忘记初始化边界条件（如 `dp[0] = 0`、`dp[0][j] = j`），导致结果错误
2. **遍历顺序错误**：0/1 背包用正序、完全背包用逆序（恰好反了），这是最常见的 DP bug
3. **状态定义模糊**：`dp[i]` 到底是前 i 个元素还是以第 i 个结尾？定义不明确导致转移方程出错
4. **溢出问题**：路径计数类问题结果很大，忘记取模；或者最优值累加时 int 溢出
5. **维度不够**：状态中遗漏关键维度，如背包问题忘记容量维、区间 DP 忘记区间长度
6. **Base Case 多余/缺失**：多余的 base case 覆盖了正常转移；或缺失导致越界

### 最佳实践

1. **先写暴力递归**：先不考虑效率，写出正确的递归解；确认正确后加记忆化；最后改写为迭代
2. **明确状态语义**：用文字精确描述 `dp[i][j]` 的含义，写在代码注释中
3. **画表验证**：手动填写小规模的 DP 表，验证转移方程正确性
4. **空间优化最后做**：先写正确的二维 DP，调试通过后再优化为一维
5. **边界测试**：用空输入、单元素、全相同、全递增/递减等边界用例验证
6. **模块化**：将常见 DP 模式（背包、LIS、区间）封装为模板，面试时快速套用

## 二十三、总结与知识图谱

```
                    动态规划（Dynamic Programming）
                              │
          ┌───────────────────┼───────────────────┐
          │                   │                   │
     核心思想              实现方式              优化技巧
     │                   │                   │
     ├─最优子结构         ├─Top-Down           ├─空间滚动
     ├─重叠子问题         │ (记忆化递归)        ├─单调队列
     └─无后效性          └─Bottom-Up          ├─斜率优化
                          (迭代填表)          ├─矩阵快速幂
                                              └─四边形不等式
          │
     分类体系
     │
     ├─线性 DP：LIS, LCS, 编辑距离, 打家劫舍
     ├─背包 DP：0/1, 完全, 多重, 分组, 混合
     ├─区间 DP：矩阵链, 石子合并, 戳气球
     ├─树形 DP：树的直径, 打家劫舍III
     ├─状压 DP：TSP, 集合覆盖
     ├─数位 DP：数字计数, 不含连续1的非负整数
     ├─概率 DP：期望步数, 随机游走
     └─博弈 DP：石子游戏, Nim
```

动态规划是算法世界的「万能钥匙」——几乎所有优化问题都可以尝试用 DP 思路建模。掌握 DP 的关键不在于背模板，而在于**训练状态定义的直觉**：面对新问题时，能够快速找到合适的状态表示和转移关系。建议按照「线性 → 背包 → 区间 → 树形 → 状压」的顺序循序渐进，每类至少刷 5-10 题形成肌肉记忆。
